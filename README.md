# dataanalyst.my

Malaysian-first resource site for data analytics, built with VitePress.

## Local development

```bash
pnpm install
pnpm docs:dev
```

## Build and preview

```bash
pnpm docs:build
pnpm docs:preview
```

## Content structure

- `docs/tools/` - tool guides
- `docs/tutorial/` - step-by-step tutorials
- `docs/data-sources/` - Malaysian data source directory
- `docs/career/` - Malaysia career guidance
- `docs/glossary/` - BM-English terms
- `docs/cheatsheets/` - quick references
- `docs/learning-paths/` - learning roadmaps
- `docs/community/` - community resources
