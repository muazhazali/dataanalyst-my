# BM-English Data Terms

| English Term | BM Term |
| --- | --- |
| Data Wrangling | Pembersihan dan Penyusunan Data |
| Feature Engineering | Kejuruteraan Ciri |
| Outlier | Nilai Terpencil |
| Forecasting | Peramalan |
| Time Series | Siri Masa |
| Data Pipeline | Saluran Data |
| Sampling | Persampelan |
| Correlation | Korelasi |
| Regression | Regresi |
| Dashboard | Papan Pemuka |

## Notes

Use English terms in code and technical documentation, but Bahasa Melayu terms are often helpful when presenting to non-technical stakeholders.
