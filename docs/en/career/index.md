# Data Analyst Career in Malaysia

## Active hiring industries

- Banking and fintech
- Telco
- GLC and digital public sector
- E-commerce and startups

## Estimated salary bands (rough)

- **Fresh grad / junior:** RM3,000 - RM5,000
- **3-5 years:** RM5,500 - RM9,000
- **Senior / lead:** RM9,000 - RM15,000+

## Most in-demand skills

1. SQL (required)
2. Dashboarding (Power BI/Tableau)
3. Python for automation and ad-hoc analysis
4. Business storytelling

## Transition path

If you are coming from an Excel-heavy role, follow this [roadmap (Malay)](/learning-paths/data-analyst-roadmap-2026) to transition into a Python + SQL workflow.
