# Start Here

If you are just getting started, follow this order to build momentum quickly:

## 1) Learn the core tools

- Start with [Python & pandas](/tools/python-pandas) or [SQL](/tools/sql).
- If reporting is your focus, continue with [Power BI](/tools/power-bi).

## 2) Build a real-world Malaysia project

- Tutorial 1: [Consumer goods price analysis (KPDN)](/tutorial/analisis-harga-barang-kpdn)
- Tutorial 2: [Malaysia population visualization (OpenDOSM)](/tutorial/visualisasi-penduduk-opendosm)

## 3) Save your key data references

- Bookmark [Malaysia Data Sources](/en/data-sources/)
- Try the API guide [OpenDOSM Quickstart](/data-sources/opendosm-api-quickstart)

## 4) Prepare your career path

- Read [Data Analyst Career in Malaysia](/en/career/)
- Pick a plan from [Data Analyst Roadmap 2026](/learning-paths/data-analyst-roadmap-2026)
