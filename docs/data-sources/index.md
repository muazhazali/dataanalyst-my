# Sumber Data Malaysia

## Tier 1 (Paling berguna)

- **data.gov.my** - katalog pelbagai kementerian/agensi
- **OpenDOSM** - populasi, buruh, CPI, perdagangan, GDP
- **BNM** - kadar tukaran, statistik monetari, OPR

## Tier 2

- EPU, Bursa Malaysia, SPR, MET Malaysia, DBKL, Penang Institute
- Community sources: Kaggle + GitHub dataset repos

## Cara guna direktori ini

1. Kenal pasti domain (ekonomi, populasi, harga, kesihatan)
2. Semak format data (CSV/API/PDF)
3. Semak kemaskini terakhir sebelum bina pipeline

Untuk API, mulakan dengan [OpenDOSM API Quickstart](/data-sources/opendosm-api-quickstart).
