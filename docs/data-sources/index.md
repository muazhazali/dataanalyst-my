# Malaysia Data Sources

## Tier 1 (Most useful)

- **data.gov.my** - catalog across multiple ministries/agencies
- **OpenDOSM** - population, labor, CPI, trade, GDP
- **BNM** - exchange rates, monetary statistics, OPR

## Tier 2

- EPU, Bursa Malaysia, SPR, MET Malaysia, DBKL, Penang Institute
- Community sources: Kaggle + GitHub dataset repos

## How to use this directory

1. Identify your domain (economy, population, prices, health)
2. Check data format (CSV/API/PDF)
3. Check the last update date before building a pipeline

For APIs, start with [OpenDOSM API Quickstart](/data-sources/opendosm-api-quickstart).
